# Gra-AI-Szachy-3D-512-p-l-8-poziom-w-wysoko-ci-po-64-pola
Gra AI Szachy 3D 512 pól to otwartoźródłowy projekt szachów przyszłości. Plansza 8×8×8 składa się z 512 pól rozmieszczonych na ośmiu poziomach wysokości, tworząc zupełnie nowy wymiar strategii. Projekt zawiera silnik AI, grafikę 3D, wersję webową oraz modułową architekturę gotową do rozwoju przez społeczność.
